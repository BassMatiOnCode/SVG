Grouping and Re-Use

G

The g element is a container element, used to group drawing elements into logical structures, and inherit common attributes (especially presentation attributes) and styles to child elements.

Use

The use element instaticates or duplicates an existing element, including its child elements. The element is referenced in the href property, position and size are determined by the x, y, width and height properties:

<use href="url(#element-id)" x="10" y="10" width="20" height="20" />

The element referenced in href can be

  • an entire internal or external SVG document
  • a document fragment
  • any internl SVG element with an id attribute, such as a symbol, g, et cetera.

Symbol

The symbol element is a template container with its own coordinate system. Elements in the symbol container are not rendered, they can be instantiated and positioned with the use element:

<svg viewBox ="-50 -50 100 100" width="100" stroke="CornflowerBlue" stroke-width="5" fill="none" style="border: none 1px silver">
    <symbol id="sy-001" viewBox="-25 -25 50 50" width="50" height="50">
        <circle cx="0" cy="0" r="22" />
        </symbol>
    <use href="#sy-001" x="0" y="0" />
    <use href="#sy-001" x="-50" y="0" />
    <use href="#sy-001" x="-50" y="-50" />
    <use href="#sy-001" x="0" y="-50" />
    </svg>

A symbol is instantiated with its upper left corner at the specified coordinates, then scaled to the specified width and height. To draw a set of concentric circles, the x and y coordinates must correlate to the width and height:

<svg viewBox ="-50 -50 100 100" width="100" stroke="CornflowerBlue" stroke-width="5" fill="none" style="border: none 1px silver">
	<symbol id="sy-001" viewBox="-25 -25 50 50" >
		<circle cx="0" cy="0" r="22" />
		</symbol>
	<use href="#sy-001" x="-7" y="-7" width="14" height="14" />
	<use href="#sy-001" x="-15" y="-15" width="30" height="30" />
	<use href="#sy-001" x="-25" y="-25" width="50" height="50" />
	<use href="#sy-001" x="-40" y="-40" width="80" height="80" />
	</svg>

Defs

The definition element defs is a container for re-usable elements. These elements are not rendered, they act as template. The elements must have a unique id attribute so that they can be referenced from use elements for instantiation.

The defs element is very similar to the g element, the difference is that a g element is rendered, a defs element not.
