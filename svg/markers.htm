Markers

First, let's draw an arrow head and get it's shape correct:

<svg width="10" height="10"">
<g stroke="none" fill="darkorange">
    <path d="M 0 0 L 10 5 0 10 z"/>
    </g>
</svg>

Here it is:

This arrow will be combined with the following line:

<svg width="50" height="50">
<line stroke="darkorange" x1="10" y1="10" x2="40" y2="40"/>
</svg>

Now let's add a marker element and combine it with a line:

<svg width="50" height="50">
<defs>
    <marker id="arrowhead" viewbox="0,0,10,10" 
        markerWidth="10" markerHeight="10" refX="0" refY="5"
        orient="auto">
        <path fill="darkorange" d="M 0 0 L 10 5 0 10 z"/>
        </marker>
    </defs>
<line x1="10" y1="10" x2="40" y2="40"
    stroke="darkorange" marker-end="url(#arrowhead)"/>
</svg>

Note that the arrow head extends the length of the line. This is ok, otherwise a wider line could interfer the arrow tip.
