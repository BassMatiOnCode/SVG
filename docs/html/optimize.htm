Website Optimizations

Reduce the Number of Files

Every resource (file), referenced from a document and it refereneced documents, must be retrieved, theoretically, from the server. However, there are some considerations, that might ease that constraint, Firstly, the browser maintains a cache. If the same resource is used multiple times (as is often the case with list style images), the request will likely be served from the local cache.

If you need many different icons, consider using a single sprite image that contains multiple tiles. SVG images are advantageous here, they can define multiple drawing elements, decorated with a class attribute which allows it to control visibility of the desired sprite image.
